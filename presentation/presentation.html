Processing Pipeline

4 phases.
Zéro blocage serveur.

Flux asynchrone strict — aucun thread bloquant, le serveur reste disponible pendant tout le traitement IA.

Phase 01
Ingestion
  • Upload /api/upload avec UUID unique
  • FFMPEG — métadonnées + thumbnail au midpoint
  • Flux SSE enregistré pour suivi temps réel
Phase 02
AI Processing
  • Whisper transcrit l'audio, horodatage mot/mot
  • YOLO détecte objets à 1fps, bounding boxes
  • EasyOCR extrait texte visuel des frames
Phase 03
Stockage
  • SQLAlchemy ORM — MySQL ou SQLite
  • Video → Segments → Words → Objects → VisualText
  • Événement 100% déverrouille la vidéo
Phase 04
Recherche
  • RapidFuzz — fautes de frappe tolérées
  • Index HashMap O(1) → timeline instantanée
  • Multi-modal : transcriptions, objets, OCR
